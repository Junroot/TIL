# 특정 필드가 없는 도큐먼트 조회 쿼리

## 목표

- 특정 필드가 없는 도큐먼트 조회 쿼리 작성법을 이해한다.

## exists

- `exists` 쿼리를 통해서 필드를 존재하는 도큐먼트를 필터링할 수 있다.

```
GET /_search
{
	"query": {
		"exists": {
			"field": "user"
		}
	}
}
```

- 아래 예시와 같이 `must_not`을 통해 필드가 없는 도큐먼트도 조회가 가능하다.

```
GET /_search
{
    "query": {
        "bool": {
            "must_not": {
                "exists": {
                    "field": "your_field"
                }
            }
        }
    }
}
```
