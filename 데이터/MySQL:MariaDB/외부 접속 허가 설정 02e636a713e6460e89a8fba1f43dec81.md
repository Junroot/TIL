# 외부 접속 허가 설정

`/etc/mysql/mysql.conf.d/mysqld.cnf` 에서 bind-address를 0.0.0.0로 수정해야된다. [localhost](http://localhost) 뿐만 아니라 모든 ip의 접속을 다 허가한다는 의미다.
