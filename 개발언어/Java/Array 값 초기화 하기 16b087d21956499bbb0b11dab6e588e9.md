# Array 값 초기화 하기

고정된 크기의 배열을 선언하고, 전부 같은 값으로 초기화 해야되는 상황이 있었다. loop문을 통해서 모든 요소를 초기화하는 방법도 있지만, `Arrays` 에 이를 이미 구현해둔 내용이 있다.

```java
Clazz[][] board;
Arrays.fill(row, new Clazz());
```
