# 우선순위 큐 사용법

`queue` 내장 모듈에 구현이 되어있다.

```python
from queue import PriorityQueue

que = PriorityQueue()
que.put(4)
que.put(1)
que.put(7)
que.put(3)

print(que.get())  # 1
print(que.get())  # 3
print(que.get())  # 4
print(que.get())  # 7
```
