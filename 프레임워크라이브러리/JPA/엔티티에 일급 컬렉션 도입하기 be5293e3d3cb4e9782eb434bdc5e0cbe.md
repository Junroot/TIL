# 엔티티에 일급 컬렉션 도입하기

```java
@Embeddable
public class RoomUsers {

    @OneToMany(mappedBy = "room")
    private final List<User> users = new ArrayList<>();
		...
}
```

```java
@Entity
public class Room {

		@Embedded
    private RoomUsers users;
		...
}
```

`@Embeddable` 과 `@Embedded` 어노테이션을 붙이면 엔티티에 일급 컬렉션을 사용할 수 있다.
